# power-consumption-calculator
2nd project in python, a simple calculator that conjectures power consumption of electrodomestics in kWh.
# language used: python
# How it calculates: 
(power consumed * time used) / 1000 * 30;
monthly cost in R$: monthly consumption * 0.85
# user instrunction:
insert the power usage amount (i.e. 51.7nkWh), then add the time the item stayed on. its as shrimple as that.
